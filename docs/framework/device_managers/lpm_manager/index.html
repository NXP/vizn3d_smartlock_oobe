Low Power Manager

The Low Power Device Manager is unique amongst the managers because it does not have the typical Init and Start functions that the other managers do. Instead, the Low Power Manager has APIs to register a device (only one at a time), configure how deep a sleep the board should enter, enable sleep mode, and more.

note

Due to the unique nature of the low power devices being an abstract "virtual" device, only one LPM device can be registered to the LPM manager at a time. However, there should be no need for more than one LPM device because other devices can configure the current low power mode states by using the Low Power Manager APIs.
